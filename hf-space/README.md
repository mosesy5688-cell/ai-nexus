# Free2AITools Discovery

Live demo of the [Free2AITools](https://free2aitools.com) catalog over the public REST API and MCP server. Search models, datasets, papers, agents, and prompts; recommend a model for a task under hardware constraints; compare up to 25 entities side-by-side; or grab a one-line MCP install snippet for Claude Desktop, Cursor, or Windsurf.

- **Website**: https://free2aitools.com
- **MCP server**: `https://free2aitools.com/api/mcp` (Streamable HTTP, JSON-RPC 2.0)
- **Install via Smithery**: https://smithery.ai/servers/mosesy5688/free2aitools
- **Source**: https://github.com/mosesy5688-cell/ai-nexus

Free, no auth, no API key.
